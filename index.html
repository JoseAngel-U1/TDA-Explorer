TDA Explorer: Visualizando Estructuras de Datos

Este proyecto sirve como una herramienta de aprendizaje visual e interactiva para entender cómo funcionan los Tipos de Datos Abstractos (TDA) y sus operaciones. Explora las estructuras con modelos 3D y animaciones.

Explorar el Glosario TDA

Tipos de Datos Abstractos (TDA)

Un TDA es un conjunto de datos al cual se asocian operaciones. Provee una interfaz para realizar las operaciones permitidas, abstraída de cómo están implementadas.

Pilas (Stacks)

Clasificación: Lineal.

Regla Clave: Sigue la regla LIFO (Last In, First Out): El último en entrar es el primero en salir.

Operaciones: Push (insertar) y Pop (eliminar), solo desde la parte superior.

Visualización Interactiva

Colas (Queues)

Clasificación: Lineal.

Regla Clave: Sigue la regla FIFO (First In, First Out): El primero en entrar es el primero en salir.

Distinción de Extremos: Se opera desde el Frente (por donde salen) y Final (por donde entran)

Visualización Interactiva

Árboles (Trees)

Clasificación: No Lineal.

Regla Clave: Estructura jerárquica. Cada nodo puede tener hijos y un único padre, excepto el nodo raíz.

Operaciones: Recorridos (preorden, inorden, postorden), Inserción, Eliminación.

Visualización Interactiva

Listas (Lists)

Clasificación: Dinámica.

Definición: Colección de elementos que se almacena de forma secuencial y puede crecer o reducirse dinámicamente.

Representación: Se construyen usando Nodos. Los nodos están enlazados mediante referencia.

Tipos: Simple (cada nodo apunta al siguiente) y Doble (cada nodo apunta al siguiente y al anterior).

Visualización Interactiva

Recursividad: La Función que se Llama a Sí Misma

Definición: Es una técnica donde una función se llama a sí misma.

Propósito: Reutilizar código y resolver un problema dividiéndolo en subproblemas más pequeños.

Regla Fundamental: Siempre debe tener una condición de parada (condición base) para evitar bucles infinitos.

Flujo de Ejecución: Trabaja a través de llamadas (bajada), y el problema se resuelve en el flujo de retornos (subida). El proceso va devolviendo resultados parciales nivel por nivel.
